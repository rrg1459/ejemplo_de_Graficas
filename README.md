# README

Ejemplo de Graficas con las gemas de chartkick y highcharts-rails


Gema:
(https://github.com/ankane/chartkick)


### Prerequisitos

Gemfile:

```
gem 'chartkick',                        '>= 5.0.4'      # Use graphics
gem 'highcharts-rails',                 '~> 6.0.3'      # Use graphics
```

application.js:
```
//= require highcharts
//= require chartkick
```
